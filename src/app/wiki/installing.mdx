This guide will help you install Qortal and set up your account.

## System Requirements
- Windows, macOS, or Linux
- At least 4GB of RAM
- 20GB of free disk space

## Windows Installation
1. Download the installer from the official website.
2. Run the `.exe` file.
3. Follow the on-screen instructions.

## macOS Installation
1. Download the `.dmg` file.
2. Open the installer and drag Qortal to Applications.
3. Run the app and follow the setup wizard.

## Linux Installation
1. Open a terminal and run:
   ```sh
   sudo apt install qortal
2. Start the Qortal application.

## Creating an Account
1. Open Qortal and click "Create Account."
2. Save your seed phrase somewhere safe.
3. Confirm your details and log in.
