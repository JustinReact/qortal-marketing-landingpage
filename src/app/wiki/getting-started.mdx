Qortal‚Äôs system requirements depend on how you plan to use the platform. If you're **not running your own node**, the requirements are minimal‚Äîon Windows, the software takes up only **264 MB of disk space**.

## System Requirements For Running Qortal Hub

- **Minimal system impact**: No significant CPU or RAM usage.
- **Disk space**: **264 MB** on Windows.
- **Internet access**: Required to connect to remote nodes.

If you choose to run a **full node**, your computer will need more resources, as it participates in securing the network and maintaining the blockchain.

### Minimum Requirements

- **CPU**: Dual-core processor
- **RAM**: **2GB or more**
- **Disk space**: **At least 50GB**, depending on usage
- **Java Version**: Java **11.0.2+** installed
- **Internet Access**: Admin access to your router is recommended for **port forwarding** and setting a static IP.

### Recommended for Optimal Performance

- **CPU**: Quad-core processor or better
- **RAM**: **8GB or more** (Windows users should have no other heavy applications running)
- **Disk Space**: **At least 50GB** for the blockchain database
- **Additional Storage**: If you plan to use **Qortal Data Network (QDN)** to host or browse websites, you may need more than **50GB**.

## Installing Qortal Hub on macOS

Follow these steps to install Qortal on macOS.

### Step 1: Download the Mac Installer

1. Visit the [official Qortal downloads page](https://qortal.dev/downloads).
2. Download **Qortal Hub** for macOS (`.dmg` file).

### Step 2: Install Qortal Hub

1. Open your **Downloads** folder and double-click the **Qortal Hub** `.dmg` file.
  <figure>
  <img
    src="/images/Wiki/QortalHubMacDownload.png"
    alt="Qortal Hub being installed on Mac"
    style={{ width: "100%", maxWidth: "600px" }}
  />
  <figcaption>Qortal Hub being installed on Mac</figcaption>
</figure>
2. Drag and drop the Qortal Hub app icon into the Applications folder to install Qortal Hub.
  <figure>
  <img
    src="/images/Wiki/QortalHubMacDrag.png"
    alt="Qortal Hub icon being dragged into the Applications folder"
    style={{ width: "100%", maxWidth: "400px" }}
  />
  <figcaption>Qortal Hub icon being dragged into the Applications folder</figcaption>
</figure>

### Step 3: Launch Qortal Hub

1. Open **Launchpad** or the **Applications** folder.
2. Click **Qortal Hub** to launch the application.
3. If MacOS warns you that the app is from an unidentified developer:
   - Click **Open** when the modal pops up.

<figure>
  <img

### Step 4: Allow Permissions (if required)

1. If prompted, grant **network access** to Qortal Hub.
2. Ensure that macOS **Firewall** settings allow Qortal to connect.

### Step 5: Start Using Qortal

1. Qortal Hub should now be running. You can access it from the menu bar or dock.

## Installing Qortal Hub on Linux

### Step 1: Download the Linux Installer

1. Visit the [official Qortal downloads page](https://qortal.dev/downloads).
2. Download **Qortal Hub** for Linux (`.AppImage` file).

<figure>
  <img
    src="/images/Wiki/LinuxDownload.png"
    alt="Downloading Qortal installer from the official website"
    style={{ width: "100%", maxWidth: "400px" }}
  />
  <figcaption>Downloading the Qortal Hub Linux installer from Qortal.dev/downloads</figcaption>
</figure>

### Step 2: Make the File Executable

Before running the installer, you need to give it execution permissions:

1. Open your **Downloads** folder.
2. Right-click the **Qortal Hub** `.AppImage` file and select **Properties**.
3. Go to the **Permissions** tab.
4. Enable **Allow executing file as program**.

<figure>
  <img
    src="/images/Wiki/LinuxAppPermission.png"
    alt="Giving the correct installer permissions on Linux"
    style={{ width: "100%", maxWidth: "400px" }}
  />
  <figcaption>Giving the correct installer permissions on Linux</figcaption>
</figure>

### Step 3: Run Qortal Hub

1. Open a terminal and navigate to your Downloads folder (or wherever you saved the file):

```javascript
cd ~/Downloads
```

2. Run the Qortal Hub installer:

```javascript
./Qortal-Hub_X.Y.Z.AppImage
```
